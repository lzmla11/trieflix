# TrieFlix

TrieFlix es una plataforma de búsqueda de películas desarrollada en C++.

El proyecto implementa una estructura basada en Tries y trigramas para permitir búsquedas rápidas por:
- palabras,
- frases,
- subpalabras,
- tags.
